# Production Grade Shopping App
A Production Grade Shopping App to learn AWS EKS, Terraform, Helm, ArgoCD and GitHub Actions.

Since we would be hosting this project on AWS, as a cleaner FinOps practice, all the resources need to have the below mandatory tags as a standard:
- Name
- CreatedOn
- CreatedBy
- CreationMode (Manual or IAC)
- Project (Always ProductionGradeShoppingApp)
